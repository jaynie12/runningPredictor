# runningPredictor
Based on a dataset taken from Kaggle, you can draw a route,enter your age and gender and receive a rough prediction as to how long it will take you to run the route.
The running predictor helps runners plot a route and displays an estimated time to complete  it, based on age and gender specific averages for the distance drawn. As a reference, the  application also shows the route's elevation and total distance.  

**Usefulness**
In terms of cost, time, or effort, this application saves users time and effort when planning 
their workouts, improving productivity.    
As a runner, planning my runs for the week is time-consuming, but this app helps me identify 
routes that fit my schedule. When preparing for races, I can evaluate my training by seeing 
how long it takes people in my age group to run the race route. 

**How to use**

1. Open up:  https://predict-running-time.streamlit.app/
2. Draw a route (for best results draw a long one as the time is rounded to the nearsest minute)
3. Enter Gender and Age
4. Click submit and recieve information on your route

**Installation steps""
1. Pip install requirements.txt
2. run command: Streamlit run MapInteractions.py
